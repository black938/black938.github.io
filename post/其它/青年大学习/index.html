1. 简介

一个简单的脚本，辅助各位团支书、班长筛选出本班没有完成青年大学习的同学，限安徽地区。

import requests
import json
import datetime
# members填写本班人员姓名
members = ['张三','李四','王五']
finished_members = []
# 这是后台一个奇葩的参数规律
reason_stage = 77+datetime.datetime.now().isocalendar()[1]*2
url = r"http://dxx.ahyouth.org.cn/api/peopleRankStage?table_name=reason_stage"+str(reason_stage)+r"&level1=%E7%9B%B4%E5%B1%9E%E9%AB%98%E6%A0%A1&level2=%E5%AE%89%E5%BE%BD%E5%86%9C%E4%B8%9A%E5%A4%A7%E5%AD%A6&level3=%E4%BF%A1%E6%81%AF%E4%B8%8E%E8%AE%A1%E7%AE%97%E6%9C%BA%E5%AD%A6%E9%99%A2&level4=17%E8%AE%A1%E7%A7%913%E7%8F%AD%EF%BC%8836%EF%BC%89"
r = requests.get(url)
result  = r.content.decode("unicode_escape")
for d in json.loads(result)["list"]["list"]:
    finished_members.append(d["username"])

unfinished_members = set(members) - set(finished_members)
unfinished_members = sorted(unfinished_members,key=lambda x:x.encode("utf-8"))
for u in unfinished_members:
    print(u)

3. 注意事项

  • members参数要换成自己班上成员的真实姓名
  • reason_stage参数的这个规律是我自己摸出来的 如果出现error 请手动调整reason_stage参数 参照当期督促人员公布的查询链接中的reason_stage ^_^
  • 仅供学习交流使用
